# Инквизитор Ордо Академия — Warhammer 40,000

Обучающая игра по высшей математике в сеттинге Warhammer 40,000.

## 📁 Структура проекта

```
gameforcollege/
├── inquisitor-academica.html  ← Главный файл игры
├── college.ini                ← База вопросов
├── img/
│   ├── inquisitor.svg         ← Портрет Инквизитора (замени на свой)
│   ├── techpriest.svg         ← Портрет Техножреца (замени на свой)
│   └── heretic.svg            ← Портрет Демона (замени на свой)
└── README.md
```

## 🎮 Как запустить

### Вариант 1: Локальный веб-сервер (РЕКОМЕНДУЕТСЯ)

Из-за ограничений безопасности браузеров, лучше использовать локальный сервер:

**Python 3:**
```bash
python -m http.server 8000
```

**Node.js (если установлен):**
```bash
npx http-server
```

Затем открой в браузере: `http://localhost:8000/inquisitor-academica.html`

### Вариант 2: Напрямую в браузере

Открой файл `inquisitor-academica.html` в **Firefox** (он лучше поддерживает локальные файлы).

**Для Chrome/Edge:** запусти браузер с флагом:
```bash
chrome.exe --allow-file-access-from-files
```

## 🎨 Замена портретов

Замени файлы в папке `img/` на свои изображения:
- `inquisitor.svg` → портрет Инквизитора (300×300px)
- `techpriest.svg` → портрет Техножреца (300×300px)
- `heretic.svg` → портрет Архидемона (300×300px)

Можешь использовать PNG/JPG — просто переименуй файлы или измени пути в коде (строки 489-508 в HTML).

## 🧠 Как играть

1. **Цель:** Ответь на все вопросы по высшей математике, сохраняя ментальную целостность.

2. **Механики:**
   - **Mental Integrity**: Твоё здоровье. Падает при неправильных ответах.
   - **XP**: Опыт. Растёт при правильных ответах.
   - **Progress**: Сколько вопросов пройдено.

3. **Персонажи:**
   - **Инквизитор** — задаёт обычные вопросы
   - **Техножрец** — появляется каждые 5 вопросов с подсказками
   - **Архидемон** — появляется каждые 5 вопросов как босс

4. **Кнопки:**
   - **ПОДТВЕРДИТЬ ОТВЕТ** — отправить ответ на проверку
   - **РАСКРЫТЬ ИСТИНУ** — показать правильный ответ (без штрафа)
   - **СЛЕДУЮЩИЙ ВОПРОС** — перейти к следующему
   - **НАЧАТЬ ЗАНОВО** — сбросить прогресс

5. **Прогресс сохраняется** автоматически в localStorage браузера.

## 📝 Формат college.ini

Вопросы хранятся в текстовом файле в формате:

```
Вопрос 1. Текст вопроса...

Ответ:
Текст ответа с формулами...
Дополнительные пояснения...

---

Вопрос 2. ...
```

Игра автоматически парсит все блоки между `---`.

## 🛠 Технические детали

- **Один HTML-файл** — всё встроено (CSS + JS)
- **Без интернета** — работает полностью offline
- **Без библиотек** — чистый Vanilla JS
- **localStorage** — сохранение прогресса
- **Адаптивный дизайн** — работает на мобильных
